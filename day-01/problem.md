Problem: Prime Numbers in Array up to Index N
📝 Problem Statement
Given a predefined array of integers:

[10, 7, 15, 3, 8, 11]
You are given an integer N.
Print all prime numbers present in the array from index 0 to N (inclusive).
📥 Input
A single integer N
📤 Output
Print all prime numbers from index 0 to N
🔒 Constraints
0 ≤ N < length of array
🧪 Sample Input
3
✅ Sample Output
7 3
